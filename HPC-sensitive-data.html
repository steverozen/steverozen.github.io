Duke School of Medicine High Performance Computing for Sensitive Data

The Duke University School of Medicine has an on premises high performance computing (HPC) cluster offered by the OASIS and the SoM Research Computing Center. See the Slurm scheduler, InfiniBand networking within the cluster, and a Lustre file system

  • 27 compute nodes (64 hyperthreaded cores, 1 TB RAM, $0.07 / CPU-hour)
  • 1 GPU node (4 X NVIDIA L40S GPUs + 64 CPU cores and 1 TB RAM, $3.50 / GPU-hour

    • To request access, technical help, consulting, or to discuss adding your own node to the HPC cluster: https://bit.ly/som-hpc-request (Duke internal web site)
