High Performance Computing for Non-Sensitive Data

DCC - Duke Compute Cluster

The Duke Office of Information Technology (OIT) provides a high-permance computing (HPC) cluster called DCC (Duke Compute Cluster, https://bit.ly/oit-dcc).

  • DCC runs AlmaLinux and the Slurm scheduler. Two Slum partitions are free to use. In addition, labs can add their own nodes (provided they are compatible with the cluster) and always get priority access on their node. Other labs can use the node but are evicted immediately if the owner wants to use the node.
  • DCC provides most bioinformatics software packages using the “Environment Modules” system (https://modules.sourceforge.net), and other standard packages can usually be added to the module system on request.
  • PIs can arrange with OIT to add their own nodes to DCC. Computing can also be reserved at $85 / core-year. Additional pricing at https://oit.duke.edu/help/articles/kb0030661/.
  • DCC offers the “Open OnDemand” web-browser interface ( https://openondemand.org). This provides a Linux desktop plus RStudio, JupyterLab notebooks, and VS Code.
  • Data storage: 1 TB free storage per lab, additional storage available for rent. Archival storage is $62 / TB-year without backup and $144 / TB-year with backup. Additional pricing for higher speed storage is at https://oit.duke.edu/help/articles/kb0030661.
  • For additional help and advice on DCC contact OIT (requires Duke MS Outlook)

The Department of Computer Science also has an HPC cluster that is only available to faculty in that department; see https://cs.duke.edu/resources/computing-resources/faqs
